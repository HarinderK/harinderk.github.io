JavaScript vs Ruby

JavaScript and Ruby are both object oriented programming languages, so they have a lot of similarities. There are place though, where they are quite different. One such concept is the ‘for’ loop.

In JavaScript, a for loop will be written like this:

for (i=0; i < n; i++) {

// code here

}

Whereas in Ruby, a for loop will be something like:

for i in 0...n

# code here

The functionality of ‘for’ loop in both languages is the same.
