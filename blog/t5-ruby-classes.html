Ruby Classes

We have had some experience with Ruby objects (integer, float, string, array etc.). The behavior and attributes of these objects are already defined in Ruby. But what if we want to create our very own object with particular attributes. Let’s say I want ruby to recognize ‘Dog’ as an object as well. Just like we do with other objects like array, I want the similar functionality.

It is possible to create your very own object in ruby using Classes. If I want to create a general Dog class, with attributes like name, age and breed, I can do so like this:

class Dog (name, age, breed)

@name = name

@age = age

@breed = breed

end

Now when I want to use Dog as an object, I can do so like this:

my_dog = Dog.new(“Max”, 3, “Lab”)

The variables, like @name, are called Instance variables. Now we have our very own defined object, but there isn’t much it can do. We can make it even more useful by adding instance methods. For example, lets add instance method to our class:

def isPuppy?

if @age <=1

return true

else return false

end

end

If I run my_dog.isPuppy?, it will return false.
