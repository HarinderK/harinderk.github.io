Arrays and Hashes

Arrays are ordered lists in which you can store a number of values inside a single variable. This will be an indexed list starting from index 0.

For example, lets define a simple array of integers:

demoArray = [4,33,6,544,66,0,9]

demoArray[0] will give us access to integer 4

demoArray[3] will give us access to integer 544

demoArray[6] will give us access to integer 9

Similarly, an array can hold string values as well as mixed values.

mixArray = ["Hello", 34, 8.6, "there"]

Hash is an unorderd list, and has key-value pairs. While in array you can access any element using the index number, in a hash you can access the value of the element through the key. You can think of hashes as a way of mapping one value to the another.

For example, lets define a hash containg owner and pets as key value pairs:

newHash = {

'Steve' => 'dog',

'Harinder' => 'cat',

'Sarah' => 'horse'

}

newHash['Steve'] will give is access to value 'dog'

We can also use symbols as hash keys, which makes the retrieving of values faster and hence, increase the efficiency of the program. For example:

myHash = {

:Harry => 'owl'

}
