Classes vs Ids

Classes and Ids are style selectors in CSS. It is way to describe your own selectors and style the content. The best practices while using Class and ID are: - ID is used when you want to define select and style one unique element - Class is used when there are bunch of elements you need to define, its more generic. - Each element can only have one ID - Same class can be used on multiple elements - You can use multiple classes to style the same element - IDs have special browser functionality â€“ hash value in the URL - An element can have both ID and Class
